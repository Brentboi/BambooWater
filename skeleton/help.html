What is Bamboo Water?

Bamboo Water is a web application that allows the user to record his/her fiber and water intake for the day.

Adding an Entry

To add an entry into the log you would press the add (+) button, which causes a window to popup. In this window you can select to add food or water. Then you can type the name of the food followed by amount and finally units (grams or liters). After this you can press the submit button which will close the modal. The new entry should appear at the top of the log.

Editing an Entry

To edit an entry on today's log you need to click on the entry. Once that happens a window should pop up with the details of that entry where you can edit the details of that entry. You can then either hit submit which will update the entry with the new details you have added, or you can press the delete button which will remove the entry from the list.

Viewing Log of a Previous Day

Viewing a log of a previous day is very simple. All you need to do is go to the history page by pressing the middle button on the menu bar. Once there you should see a list with different dates.
